* 구글 메인 페이지 클론 코딩을 가르치고 있는데, 거의 다 끝나간다.
* 이제 웬만한 태그는 이전 코드를 베껴쓰지 않아도 잘 쓰게 된 것 같다. 가르친 보람이 있다.
* 대신 아직 CSS 부분이 익숙하지 않아 애를 많이 먹는 것 같다. 좀 더 연습이 필요해 보인다.
